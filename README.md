# Pig-Game
Pig Game is a game of rolling a dice. It can be played by one or two users. In this game, the first player to reach a score of 100 wins the game. On rolling the dice, the possibly of rolling a one(1) is 1/6. On rolling a 1, the current score is lost and the second player gets a chance of rolling the dice.
